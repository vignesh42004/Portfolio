Predictive Maintenance for Rotating Machines using TinyML

This project uses a TinyML model on an ESP32 microcontroller and an ADXL345 vibration sensor to predict faults in rotating machinery by detecting subtle changes in vibration patterns. This predictive maintenance approach helps in timely maintenance and prevents machine failure

A Dual Rotor Wind Turbine: Upwind and Downwind Turbine Integration

This project focuses on the integration of dual rotors (Upwind and Downwind) in wind turbines to significantly increase power output and enhance wind resource utilization. By strategically aligning the rotors with the wind direction, the system inherently reduces a source of parasitic power loss, resulting in a net gain in overall power generation.
